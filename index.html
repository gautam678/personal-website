Mlaas
An open source framework that can be used to train models and run machine learning algorithms on the cloud. This is targeted at people with limited experience and resources. We provide computing resources to enable users to easily setup models and expose these for use in external real-time applications. We were able to offload the computational and management heavy-lifting to the cloud and make machine learning accessible to a large demographic that can benefit from it.

Raspberry-Vision
A prototype that uses Raspberry Pi and a Pi camera to identify objects that are usually inside a house. The idea behind this project is to build a product that can be used by virtually impaired people to navigate through their house and work with objects without colliding with them. For a prototype, images of a bottles were trained so anyone wearing the devide could interact with them.
